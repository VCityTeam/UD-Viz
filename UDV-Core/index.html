UDV Examples

UDV is a JavaScript client based on iTowns allowing to visualize, analyse and interact with urban data. It is developed as part of the Virtual City Project.

This page lists the demos and examples of UDV.

Demos

List of demonstrations combining modules and / or extensions of UDV.

  • UDV Full modules Demo

    Demonstrate the modules Authentication and Validation.

    This demo implements all the modules and extensions currently available. As of today, this includes :
    - Temporal
    - Help & About
    - Documents
    - Contribute
    - Guided tours
    - Authentication
    - Documents to validate
    - Document comments
    - Geocoding
    - 3DTiles debug
    - Camera positioner
    - Document links

Individual Modules and Extensions Demos

Each individual module and extension should have an associated example. They are listed here.

  • Documents

    Demonstrate the Documents Module.

    In this demo you can consult documents in a document browser and search them using keywords and other fields.

  • Guided Tour

    Demonstrate the GuidedTour Module.

    In this demo you can try the mode guided tour which is a virtual navigation in a 4D (3D + time) and documented representation of the city.

  • Contribute

    Demonstrate the Contribute Extension.

    In this demo you can try the Contribute extension which allows to create, update and delete documents. You can also consult documents in a document browser and search them using keywords and other fields.

  • Authentication

    Demonstrate the Authentication Extension.

    This demo allows you to create a user account, log in and log out. This demonstrates how user information can be retained through the session.

  • Documents to validate

    Demonstrate the Documents to validate Extension.

    In this demo, the creation of a document actually adds a "document to validate", which is a contribution request. These documents can be udpated, deleted, commented and validated. When validated, the actual document is created. This demo makes use of the authentication module : only the user who made the contribution and the admin can see them; and only an admin can validate them.

  • Geocoding

    Demonstrate the Geocoding Extension.

    Geocoding allows a user to navigate in the city by searching addresses or place names.

  • Camera Positioner

    Demonstrate the Camera Postioner tool.

    The camera positioner is a tool that show the coordinates and the orientation of the camera.

  • 3DTiles Debug

    Demonstrate the 3DTiles Debug tool.

    3DTiles Debug displays a window which shows various information about 3DTiles objects and city objects.

  • Links

    Demonstrate the Links Extension.

    This extension adds the possibility to see the links associated to a document, as well as to create new links with city objects.
